Otorgamiento

¿Qué se necesita para sacar la libreta sanitaria por primera vez?
¿Qué se necesita para sacar la libreta sanitaria por primera vez?

La libreta sanitaria certifica el estado de salud de los trabajadores para algunas enfermedades de transmisión alimentaria, infecciosa y/o sexual a efecto de evitar su propagación, realizando el diagnostico precoz y el tratamiento oportuno del trabajador.

Requisitos:

  • Examen médico completo (clínico/a y bioquímico/a)
  • Fotocopia DNI (ambos lados)
  • Constancia de domicilio (en caso que no figure el domicilio real en el DNI)
  • Una foto tipo carnet 4×4
  • Realizar curso de BPM (Buenas Prácticas de Mano Factura) – POES – MIP

 

La participación del curso de BPM es OPTATIVA, el examen es OBLIGATORIO.

Renovación

¿Qué se necesita para renovar la libreta sanitaria?
¿Qué se necesita para renovar la libreta sanitaria?

Requisitos para renovación:

  • Examen médico completo (clínico/a y bioquímico/a)
  • Libreta sanitaria anterior
  • Realizar curso BPM (Buenas Prácticas de Mano Factura) – POES – MIP

 

La participación del curso de BPM es OPTATIVA, el examen es OBLIGATORIO.
